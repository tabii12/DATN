==================================================
PROJECT: TRAVEL BOOKING SYSTEM
API DOCUMENTATION
BASE DOMAIN: https://db-datn.onrender.com
==================================================

I. GIỚI THIỆU
--------------------------------------------------
Hệ thống Travel Booking System cung cấp API cho:
- Quản lý người dùng
- Quản lý tour
- Quản lý khách sạn
- Đặt tour
- Review
- Blog
- Lịch trình
- Dịch vụ đi kèm

Chuẩn RESTful API.
Dữ liệu trả về dạng JSON.

II. PHÂN QUYỀN HỆ THỐNG
--------------------------------------------------
1. ADMIN
   - Tạo / cập nhật / xoá tour
   - Quản lý hotel
   - Quản lý blog
   - Quản lý service
   - Duyệt booking

2. USER
   - Xem tour / hotel
   - Đặt tour
   - Viết review
   - Cập nhật thông tin cá nhân

MODULE: TOUR API
BASE URL: https://db-datn.onrender.com/api/tours
==================================================

1. Tạo Tour
--------------------------------------------------
Method: POST
Endpoint: /create
Full URL: https://db-datn.onrender.com/api/tours/create
Mô tả: Tạo mới một tour du lịch.
Content-Type: multipart/form-data
Upload: images (tối đa 5 ảnh)
Middleware sử dụng: multer

2. Lấy Danh Sách Tour
--------------------------------------------------
Method: GET
Endpoint: /
Full URL: https://db-datn.onrender.com/api/tours/
Mô tả: Lấy danh sách tất cả các tour.
Content-Type: application/json

3. Lấy Chi Tiết Tour Theo Slug
--------------------------------------------------
Method: GET
Endpoint: /detail/:slug
Full URL: https://db-datn.onrender.com/api/tours/detail/{slug}
Mô tả: Lấy thông tin chi tiết của tour dựa vào slug.
Content-Type: application/json

4. Cập Nhật Tour
--------------------------------------------------
Method: PUT
Endpoint: /update/:slug
Full URL: https://db-datn.onrender.com/api/tours/update/{slug}
Mô tả: Cập nhật thông tin tour.
Content-Type: multipart/form-data
Upload: images (tối đa 5 ảnh, không bắt buộc)

5. Xoá Ảnh Của Tour
--------------------------------------------------
Method: DELETE
Endpoint: /image/:imageId
Full URL: https://db-datn.onrender.com/api/tours/image/{imageId}
Mô tả: Xoá một ảnh cụ thể của tour theo imageId.
Content-Type: application/json

6. Cập Nhật Trạng Thái Tour
--------------------------------------------------
Method: PATCH
Endpoint: /status/:slug
Full URL: https://db-datn.onrender.com/api/tours/status/{slug}
Mô tả: Cập nhật trạng thái tour (ví dụ: hoạt động / ngừng hoạt động).
Content-Type: application/json
